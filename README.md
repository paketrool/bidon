# Бидон

`Бидон` — учебный DSL-язык с синтаксисом на русском подростковом сленге.
Идея: писать код так, как говоришь с друзьями, но без хаоса и с понятной логикой.

## 1) Описание языка

- ЦА: начинающие программисты (подростки)
- Стиль: мемно и дружелюбно, но без перегруза узким сленгом
- Модель исполнения: **интерпретируемый язык**
- Реализация: Python (`slanglang/interpreter.py`)

### Словарь конструкций

| Классика | Бидон | Смысл |
|---|---|---|
| variable declaration | `заведи` | Объявить переменную |
| assignment | `=` | Присвоить новое значение |
| if | `если_чё (...) { ... }` | Ветвление |
| else | `иначе { ... }` | Альтернативная ветка |
| while | `го_пока (...) { ... }` | Цикл пока условие истинно |
| for range | `го_по i от 1 до 5 { ... }` | Счетный цикл (включительно) |
| function | `флекс имя(...) { ... }` | Определить функцию |
| return | `верни` | Вернуть значение |
| print | `чекни` | Вывести в консоль |
| true / false | `тру` / `фолс` | Булевы литералы |

## 2) Синтаксис

### Переменные

```bidon
заведи имя = "Саня";
заведи возраст = 15;
```

### Присваивание

```bidon
возраст = возраст + 1;
```

### Условия

```bidon
если_чё (возраст >= 14 && возраст < 18) {
    чекни "подростковый режим";
} иначе {
    чекни "другой режим";
}
```

### Циклы

```bidon
го_пока (x > 0) {
    чекни x;
    x = x - 1;
}

go_по i от 1 до 3 {
    чекни i;
}
```

### Функции

```bidon
флекс квадрат(n) {
    верни n * n;
}

чекни квадрат(4);
```

## 3) Грамматика (упрощенный EBNF)

```ebnf
program         = { statement } ;

statement       = varDecl
                | assignStmt
                | printStmt
                | ifStmt
                | whileStmt
                | forStmt
                | funcDef
                | returnStmt
                | exprStmt ;

varDecl         = "заведи" IDENT "=" expression ";" ;
assignStmt      = IDENT "=" expression ";" ;
printStmt       = "чекни" expression ";" ;
ifStmt          = "если_чё" "(" expression ")" block [ "иначе" block ] ;
whileStmt       = "го_пока" "(" expression ")" block ;
forStmt         = "го_по" IDENT "от" expression "до" expression block ;
funcDef         = "флекс" IDENT "(" [ params ] ")" block ;
params          = IDENT { "," IDENT } ;
returnStmt      = "верни" [ expression ] ";" ;
exprStmt        = expression ";" ;
block           = "{" { statement } "}" ;

expression      = logicOr ;
logicOr         = logicAnd { "||" logicAnd } ;
logicAnd        = equality { "&&" equality } ;
equality        = comparison { ("==" | "!=") comparison } ;
comparison      = term { ("<" | "<=" | ">" | ">=") term } ;
term            = factor { ("+" | "-") factor } ;
factor          = unary { ("*" | "/" | "%") unary } ;
unary           = ("-" | "!") unary | call ;
call            = primary { "(" [ args ] ")" } ;
args            = expression { "," expression } ;
primary         = NUMBER | STRING | "тру" | "фолс" | IDENT | "(" expression ")" ;
```

## 4) Правила парсинга

Интерпретатор использует классический пайплайн:

1. Лексер (`tokenize`): разбивает текст на токены (ключевые слова, идентификаторы, числа, строки, операторы).
2. Парсер (recursive descent): строит AST по приоритетам операторов.
3. Исполнитель (`Interpreter`): обходит AST, хранит значения в областях видимости (`Environment`).

Ключевые детали:

- Блоки `{ ... }` создают вложенную область видимости.
- Функции (`флекс`) поддерживают параметры и `верни`.
- `го_по` работает включительно (`от` и `до` входят в диапазон).
- Ошибки синтаксиса и рантайма выносятся с понятными текстами.

## 5) Код интерпретатора

- Точка входа CLI: `bidon.py`
- Ядро: `slanglang/interpreter.py`

Запуск:

```bash
python bidon.py examples/01_hello_world.bidon
```

## 6) Примеры программ

### Hello world

```bidon
чекни "Хеллоу, мир! Это Бидон.";
```

### Калькулятор

```bidon
заведи a = 12;
заведи b = 5;
чекни "sum=" + (a + b);
чекни "diff=" + (a - b);
чекни "mul=" + (a * b);
чекни "div=" + (a / b);
```

### Цикл с выводом

```bidon
go_по i от 1 до 5 {
    чекни "круг " + i;
}
```

### Условная логика

```bidon
заведи возраст = 15;
если_чё (возраст >= 14 && возраст < 18) {
    чекни "Ты в подростковом режиме";
} иначе {
    чекни "Другой вайб";
}
```

### Функция

```bidon
флекс квадрат(x) {
    верни x * x;
}

чекни квадрат(9);
```

## 7) Структура проекта

- `bidon.py` — запуск интерпретатора
- `slanglang/interpreter.py` — лексер, парсер, runtime
- `examples/*.bidon` — демо программы

## Мемно, но не слишком

В языке есть юмор (`если_чё`, `чекни`, `флекс`), но базовая логика не страдает:

- ключевые слова стабильные и понятные
- операторы стандартные
- синтаксис предсказуемый

Итог: прикольно, но не кринжово для обучения.
